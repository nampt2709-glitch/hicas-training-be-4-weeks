  FILE: LogAnalyzer/Services/FileReaderService.cs (interface: Interfaces/IFileReader.cs)
  MỤC ĐÍCH: Đọc toàn bộ nội dung một file văn bản vào bộ nhớ (một string)
================================================================================

TẠI SAO CÓ FILE NÀY?
  Chương trình phân tích log đọc HẾT file một lần thành một chuỗi lớn, rồi dùng
  Regex và thuật toán đếm trên chuỗi đó. FileReaderService triển khai IFileReader — quanh
  File.ReadAllText / File.ReadAllTextAsync. BenchmarkRunnerService inject IFileReader
  so sánh ReadSync với ReadAsync.


--------------------------------------------------------------------------------
CLASS: FileReaderService (triển khai IFileReader)
--------------------------------------------------------------------------------

 PHƯƠNG THỨC: ReadSync(string path)
  - Đầu vào: đường dẫn file (path).
  - Đầu ra: toàn bộ nội dung file dạng string.
  - Cách hoạt động: gọi File.ReadAllText(path).
  - "Đồng bộ": luồng đang chạy phải CHỜ đọc xong mới chạy tiếp lệnh sau.


 PHƯƠNG THỨC: ReadAsync(string path)
  - Đầu vào: path.
  - Đầu ra: Task<string> - hứa sẽ có chuỗi khi đọc xong.
  - Cách hoạt động: gọi File.ReadAllTextAsync(path).
  - Dùng cho các nhánh "Async + ..." trong BenchmarkRunner.


LIÊN KẾT
  BenchmarkRunnerService dùng IFileReader trước khi Analyzer/Counter xử lý.
